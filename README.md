# numberspace

version 1.3.1 of a playable sudoku game for arbitrary board shapes

### features
- two puzzles using the regular set of symbols for Sudoku: the digits 1-9
- four puzzles using new symbol sets in which the symbol orientation matters

### next steps
- give the user a way to save their progress
- develop a mathematical approach to puzzle solving and generation
- redesign the graphics algorithms
